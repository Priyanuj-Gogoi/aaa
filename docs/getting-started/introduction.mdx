In this page & the following sub-pages, you will learn the basics to get started with *Bot Designer For Discord*.

### Installation
Choose your platform and follow the steps described below:

<Tabs>

<TabItem value="android" label="Android">
  <ol>
    <li>Open <a href="https://play.google.com/" title="Google Play Store">Google Play Store</a> app.</li>
    <li>Type "<strong>Bot Designer For Discord"</strong> in the search bar.</li>
    <li>Click the "<strong>Install</strong>" button in the <a href="https://play.google.com/store/apps/details?id=com.jakubtomana.discordbotdesinger" title="BDFD App">app</a> page.</li>
    <br/>
    <img src="https://user-images.githubusercontent.com/95774950/208600770-0241cf88-80d7-47a6-9ee4-fceb81d71877.jpg" alt="BDFD Play Store Install" title="BDFD Play Store Install"/>
  </ol>

:::info

You need minimum Android 5.0+ version in order to run the app.

:::
</TabItem>
